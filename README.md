Дипломная балакаврская работа в СПбГЭТУ "ЛЭТИ" по направлению "Информатика и вычислительная техника"

Разработанное приложение расчитывает передаточные характеристики заданной на вход пользователем цепи. 

## Ключевые особенности
* Авторизация пользователей при помощи access и refresh токенов. Они хранятся в cookie-наборах с флагом http-only. 
* Для активации аккаунта необходимо подтвердить почту.
* Данные о схемах пользователей хранятся в базе данных(PostgreQSL + Prisma).
* API реализовано на основе технологии GraphQL.
* Результаты расчёта передаются при помощи специального типа Subscriptions, который работает по WebSockets.
* Для поддержки всех типов запросов развёрнуто два сервера (один для Mutation и Query, второй для Subsctiptions).
* Для форм и их валидации используются собственные кастомные хуки.
* Для отрисовки графики используется пакет Chart.js.
* Данные отправляются только по протоколам HTTPS и WSS.
* Работа класса вычислений протестирована при помощи Jest.
* Параметры для элементов схемы передаются в query-параметрах запроса.
